About the Project

Bangalore House Price Predictor

This project was built as a complete end-to-end Machine Learning web application. It demonstrates the full lifecycle of a data science project — from data enhancement and cleaning, feature engineering, training 8 ML algorithms, saving the best model, and deploying with a Flask backend and modern responsive frontend.

Backend
  • Python 3
  • Flask Web Framework
  • Pandas & NumPy
  • Pickle Serialization
Frontend
  • HTML5 & Vanilla CSS
  • JavaScript (Fetch API)
  • Google Fonts (Inter)
  • Responsive Design
Machine Learning
  • Scikit-Learn
  • Random Forest (Best)
  • StandardScaler
  • 8 Algorithms Compared
Project Files
  • Bengaluru_House_Data.csv
  • best_house_price_model.pkl
  • enhanced_house_data.csv
  • model_comparison_results.csv

Project Pipeline — 6 Phases

✓

Phase 1 — Data Enhancement

Added has_gym, has_swimming_pool, has_ground, ground_area_sqft, and amenities_count to 13,320 properties using realistic probability logic based on price tiers.

✓

Phase 2 — Data Cleaning & Exploration

Removed 75 duplicates, handled 6,201 missing values via median imputation, detected outliers via IQR method. Generated 5 sets of dark-themed visualizations (histograms, box plots, heatmap, scatter plots, bar charts).

✓

Phase 3 — Feature Engineering & Preprocessing

Derived 5 new features: total_rooms, room_per_area, has_multiple_amenities, area_per_bedroom, high_amenity_property. Applied StandardScaler. Split 80/20 (10,580 train / 2,645 test).

✓

Phase 4 — Model Training & Comparison

Trained and compared 8 algorithms: Linear Regression, Ridge, Lasso, Decision Tree, Random Forest (Best — R²=60.5%), Gradient Boosting, SVR, and AdaBoost.

✓

Phase 5 — Save Best Model

Saved best_house_price_model.pkl, scaler.pkl, feature_names.pkl, label_encoder_location.pkl, and model_info.txt to the models/ directory.

✓

Phase 6 — Flask Web Application

Built this full dashboard UI with 5 pages: Dashboard, Predict Price, Data Insights, Model Info, and About. AJAX-based prediction with live results panel.
